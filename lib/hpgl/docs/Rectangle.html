The Rectangle class represents an abstrat rectangle object which posesses a width, a height and a position (x, y).

Parameters:
Name Type Description
x Number

Position of the rectangle's top-left corner along the x axis.
y Number

Position of the rectangle's topl-left corner along the y axis.
width Number

Width of the rectangle.
height Number

Height of the rectangle.
